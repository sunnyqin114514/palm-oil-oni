▍这台后端是干什么的

它是整个项目的「数据中转站」。负责读取棕榈油价格、厄尔尼诺指数、降水、产量等数据， 整理成统一格式，再交给 3D 网页或预测模型使用。你不用手动开 CSV，向它「要数据」即可。

② 预测某月棕榈油产量

综合历史天气、厄尔尼诺指数、降水、温度和西马温降交互，预测马来西亚棕榈油总产量。
展开：计算公式与原理
单产 = 截距 + 趋势项 + βONI×ONI(滞后12月) + β降水×3月滚动降水距平 + β温度×10月滚动气温距平 + β交互×西马(10月温度距平×10月降水距平) + 季节月份项
总产量 = 单产 × 最新成熟种植面积
天气数据优先用 ECMWF 季节预报，缺失则回退历史同期均值（基准情形）。
因素对单产的贡献 (吨/公顷)
选择月份，点击预测
